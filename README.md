# Support - Gather

### Login
Log in to access Gather and connect with others through hangouts. You can log in using your:
* Apple account
* Google account
Or create a user using
* email and password

### Privacy
Gather lets you present yourself through a personal profile and meet others and friends through hangouts. This necessitates some sharing of personal information. It is therefore important for us at Gather that you can trust how we handle your personal information. You can read how we handle your information here [Privacy Policy](https://github.com/OpraEria/privacy-policy)
